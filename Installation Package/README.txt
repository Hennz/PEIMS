Installation Guide

1. Double click on the Enterprise Manager Windows Installer Package.
2. Go to next. 
3. Accept the licensce aggreement and click next 
3. Then click install.
4. After Installation You can Choose to launch program.




To work with existing Database

1. Copy the pharma folder provided with MySQL files to MySQL/Data directory
2. If you dont have MySQL please download Wamp Server of MySql Standalone and copy the files provided to the specified directory


NOTE: Enterprise Manager data cannot be accessed if the following steps are not done.
